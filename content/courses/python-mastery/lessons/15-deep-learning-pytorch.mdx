## Why Neural Networks Work

The **Universal Approximation Theorem** (Cybenko, 1989; Hornik, 1991) guarantees that a neural network with a single hidden layer containing enough neurons can approximate any continuous function on a compact subset of Rⁿ to arbitrary precision. This is the mathematical foundation.

In practice, networks succeed because of **three ideas working together:**

1. **Differentiable composition** — every operation in the network (matmul, ReLU, softmax) has a computable gradient. This means we can propagate error signals backward through the entire network.

2. **Stochastic gradient descent** — we don't need the optimal parameter update at each step; a noisy direction that's *mostly correct* is enough. Given enough steps, it converges.

3. **Efficient computation** — GPUs can perform thousands of matrix multiplications in parallel. A modern GPU can compute 20+ teraFLOPs per second.

The actual mechanics: a network is a parameterized function `y = f(x; θ)`. We define a **loss** `L(y, y_true)` measuring how wrong the prediction is. We compute `∂L/∂θ` for every parameter θ (via backpropagation), then subtract a small fraction of that gradient from θ. Repeat millions of times.

## The Mathematics of Backpropagation

Consider a 3-layer network:

```
x → [W₁, b₁] → ReLU → [W₂, b₂] → ReLU → [W₃, b₃] → sigmoid → ŷ → L
```

Forward pass:
```
z₁ = W₁x + b₁
a₁ = ReLU(z₁)
z₂ = W₂a₁ + b₂
a₂ = ReLU(z₂)
z₃ = W₃a₂ + b₃
ŷ  = sigmoid(z₃)
L  = BCE(ŷ, y)
```

Backward pass (chain rule applied right to left):
```
∂L/∂z₃  = ŷ - y                    (BCE + sigmoid derivative cancel cleanly)
∂L/∂W₃  = (∂L/∂z₃) · a₂ᵀ / m       (m = batch size)
∂L/∂b₃  = mean(∂L/∂z₃)
∂L/∂a₂  = W₃ᵀ · (∂L/∂z₃)
∂L/∂z₂  = (∂L/∂a₂) · ReLU'(z₂)     (ReLU' = 1 if z>0 else 0)
∂L/∂W₂  = (∂L/∂z₂) · a₁ᵀ / m
...and so on back to W₁
```

**Key insight:** each layer only needs the gradient from the layer *above* it. No layer needs to know about any other layer. This is why deep learning frameworks (PyTorch, JAX, TensorFlow) can handle architectures of arbitrary depth — backprop is just a recursive application of the chain rule.

# Different problems require different distance metrics.

def mse(y_pred, y_true):
    """Mean Squared Error — penalizes large errors quadratically."""
    return np.mean((y_pred - y_true) ** 2)

def mse_grad(y_pred, y_true):
    return 2 * (y_pred - y_true) / len(y_true)

def mae(y_pred, y_true):
    """Mean Absolute Error — robust to outliers (linear penalty)."""
    return np.mean(np.abs(y_pred - y_true))

def huber(y_pred, y_true, delta=1.0):
    """Huber loss — MSE for small errors, MAE for large errors."""
    diff = np.abs(y_pred - y_true)
    return np.where(diff <= delta,
                    0.5 * diff**2,
                    delta * diff - 0.5 * delta**2).mean()

# ── Classification losses ──────────────────────────────────────────────────────
def binary_cross_entropy(y_pred, y_true):
    """BCE — for binary classification (sigmoid output)."""
    y_pred = np.clip(y_pred, EPS, 1 - EPS)
    return -np.mean(y_true * np.log(y_pred) + (1 - y_true) * np.log(1 - y_pred))

def bce_grad(y_pred, y_true):
    """Gradient of BCE with respect to sigmoid output (combined = ŷ - y)."""
    return y_pred - y_true

    """Softmax — converts logits to probability distribution (sums to 1)."""
    # Subtract max for numerical stability: prevents exp overflow
    e_x = np.exp(x - x.max(axis=-1, keepdims=True))
    return e_x / e_x.sum(axis=-1, keepdims=True)

def cross_entropy(logits, y_true_onehot):
    """CCE — for multi-class classification (softmax output)."""
    probs = softmax(logits)
    return -np.mean(np.sum(y_true_onehot * np.log(probs + EPS), axis=1))

def cross_entropy_grad(logits, y_true_onehot):
    """Gradient of CCE combined with softmax = (softmax(logits) - y_true) / m."""
    return (softmax(logits) - y_true_onehot) / len(y_true_onehot)

<PythonPlayground code={String.raw`import numpy as np

class NeuralNetwork:
    """
    Fully connected neural network with:
    - Configurable architecture: any depth and width
    - He initialization for ReLU layers
    - Mini-batch gradient descent
    - Gradient clipping to prevent explosion
    - Training history tracking
    """

    def __init__(self, layer_sizes: list[int], lr: float = 0.01, clip: float = 5.0):
        """
        layer_sizes: [input_dim, hidden1, hidden2, ..., output_dim]
        Example: [2, 64, 32, 1] — 2 input features, 2 hidden layers, 1 output
        """
        self.lr = lr
        self.clip = clip
        rng = np.random.default_rng(42)

        self.W: list[np.ndarray] = []
        self.b: list[np.ndarray] = []

        for i in range(len(layer_sizes) - 1):
            n_in  = layer_sizes[i]
            n_out = layer_sizes[i + 1]

            # He initialization: var = 2/fan_in
            # Why: with ReLU, ~half the neurons are inactive, so effective fan_in ≈ fan_in/2
            # Keeping variance at 1 through layers prevents vanishing/exploding signals
            scale = np.sqrt(2.0 / n_in)
            self.W.append(rng.normal(0, scale, (n_in, n_out)))
            self.b.append(np.zeros(n_out))

    def _relu(self, x: np.ndarray) -> np.ndarray:
        return np.maximum(0, x)

    def _relu_grad(self, x: np.ndarray) -> np.ndarray:
        return (x > 0).astype(np.float64)

    def _sigmoid(self, x: np.ndarray) -> np.ndarray:
        return 1.0 / (1.0 + np.exp(-np.clip(x, -500, 500)))

    def forward(self, X: np.ndarray) -> np.ndarray:
        """
        X: shape (n_samples, n_features)
        Returns: shape (n_samples, n_output)

        Stores pre-activations (z) and activations (a) for backprop.
        """
        self._z: list[np.ndarray] = []   # Pre-activations: z = Wa + b
        self._a: list[np.ndarray] = [X]  # Activations: a = activation(z)

        A = X
        n_layers = len(self.W)

        for i in range(n_layers):
            Z = A @ self.W[i] + self.b[i]   # (n_samples, n_out)
            self._z.append(Z)

            if i < n_layers - 1:
                A = self._relu(Z)            # Hidden layers: ReLU
            else:
                A = self._sigmoid(Z)         # Output: sigmoid for binary

            self._a.append(A)

        return A   # shape: (n_samples, 1) for binary classification

    def backward(self, y: np.ndarray) -> dict[str, float]:
        """
        y: shape (n_samples,) — true labels

        Returns gradient norms for monitoring.
        """
        m = len(y)
        y_col = y.reshape(-1, 1)

        # Output layer gradient: d(BCE)/d(sigmoid output) combined = ŷ - y
        # This elegant result: derivative of BCE w.r.t. sigmoid input cancels the sigmoid derivative
        dA = self._a[-1] - y_col   # (n_samples, 1)

        grad_norms = {}
        n_layers = len(self.W)

        for i in reversed(range(n_layers)):
            Z = self._z[i]
            A_prev = self._a[i]

            if i < n_layers - 1:
                # Hidden layers: apply ReLU gradient
                dZ = dA * self._relu_grad(Z)
            else:
                # Output layer: sigmoid gradient already incorporated above
                dZ = dA

            # Gradient with respect to weights and biases
            dW = A_prev.T @ dZ / m
            db = dZ.mean(axis=0)

            # Gradient clipping: prevents explosion in deep nets
            dW_norm = np.linalg.norm(dW)
            if dW_norm > self.clip:
                dW = dW * (self.clip / dW_norm)

            grad_norms[f"layer_{i}"] = dW_norm

            # Gradient for the previous layer (chain rule)
            dA = dZ @ self.W[i].T

            # Parameter update (gradient descent)
            self.W[i] -= self.lr * dW
            self.b[i] -= self.lr * db

        return grad_norms

    def train(
        self,
        X: np.ndarray,
        y: np.ndarray,
        epochs: int = 1000,
        batch_size: int = 32,
        val_data: tuple | None = None,
        print_every: int = 200,
    ) -> dict:
        """
        Mini-batch stochastic gradient descent.
        Returns training history.
        """
        n = len(X)
        rng = np.random.default_rng(0)
        history = {"train_loss": [], "val_loss": [], "train_acc": [], "val_acc": []}

        for epoch in range(epochs):
            # Shuffle training data each epoch
            idx = rng.permutation(n)
            X_shuf, y_shuf = X[idx], y[idx]

            # Mini-batch updates
            for start in range(0, n, batch_size):
                X_batch = X_shuf[start:start + batch_size]
                y_batch = y_shuf[start:start + batch_size]
                self.forward(X_batch)
                self.backward(y_batch)

            # Track metrics on full dataset
            y_pred_full = self.forward(X)[:, 0]
            eps = 1e-8
            train_loss = -np.mean(
                y * np.log(y_pred_full + eps) +
                (1 - y) * np.log(1 - y_pred_full + eps)
            )
            train_acc = ((y_pred_full > 0.5).astype(int) == y).mean()
            history["train_loss"].append(train_loss)
            history["train_acc"].append(train_acc)

            if val_data:
                X_v, y_v = val_data
                y_pred_v = self.forward(X_v)[:, 0]
                val_loss = -np.mean(
                    y_v * np.log(y_pred_v + eps) +
                    (1 - y_v) * np.log(1 - y_pred_v + eps)
                )
                val_acc = ((y_pred_v > 0.5).astype(int) == y_v).mean()
                history["val_loss"].append(val_loss)
                history["val_acc"].append(val_acc)

            if epoch % print_every == 0:
                val_str = ""
                if val_data:
                    val_str = f"  val_loss={history['val_loss'][-1]:.4f}  val_acc={history['val_acc'][-1]:.3f}"
                print(f"Epoch {epoch:4d}: train_loss={train_loss:.4f}  train_acc={train_acc:.3f}{val_str}")

        return history

    def predict(self, X: np.ndarray, threshold: float = 0.5) -> np.ndarray:
        return (self.forward(X)[:, 0] > threshold).astype(int)

    def predict_proba(self, X: np.ndarray) -> np.ndarray:
        return self.forward(X)[:, 0]

## Optimizers: SGD, Momentum, RMSProp, Adam

<PythonPlayground code={String.raw`import numpy as np

# ── Loss surface: Rosenbrock function (classic optimizer benchmark) ────────────
# Rosenbrock: f(x,y) = (1-x)^2 + 100*(y-x^2)^2
# Narrow curved valley — SGD struggles, adaptive methods shine

def rosenbrock(w):
    x, y = w[0], w[1]
    return (1 - x)**2 + 100 * (y - x**2)**2

def rosenbrock_grad(w):
    x, y = w[0], w[1]
    dx = -2*(1-x) - 400*x*(y - x**2)
    dy = 200*(y - x**2)
    return np.array([dx, dy])

# ── Optimizer implementations ──────────────────────────────────────────────────

def vanilla_sgd(grad_fn, w_init, lr=0.001, n_steps=500):
    w = w_init.copy()
    history = [rosenbrock(w)]
    for _ in range(n_steps):
        g = grad_fn(w)
        w = w - lr * g
        history.append(rosenbrock(w))
    return w, history

def momentum_sgd(grad_fn, w_init, lr=0.001, momentum=0.9, n_steps=500):
    """Momentum: accumulates gradient history. Like a ball rolling downhill.
    Accelerates in consistent gradient directions.
    Dampens oscillations in noisy/curved surfaces.
    """
    w = w_init.copy()
    v = np.zeros_like(w)   # Velocity
    history = [rosenbrock(w)]
    for _ in range(n_steps):
        g = grad_fn(w)
        v = momentum * v - lr * g    # Accumulate: keeps fraction of previous velocity
        w = w + v
        history.append(rosenbrock(w))
    return w, history

def rmsprop(grad_fn, w_init, lr=0.01, beta=0.9, eps=1e-8, n_steps=500):
    """RMSProp: adaptive learning rate per parameter.
    Divides by running RMS of past gradients.
    Parameters with consistently large gradients get smaller effective LR.
    """
    w = w_init.copy()
    v = np.zeros_like(w)   # Running average of squared gradients
    history = [rosenbrock(w)]
    for _ in range(n_steps):
        g = grad_fn(w)
        v = beta * v + (1 - beta) * g**2
        w = w - lr * g / (np.sqrt(v) + eps)
        history.append(rosenbrock(w))
    return w, history

def adam(grad_fn, w_init, lr=0.05, beta1=0.9, beta2=0.999, eps=1e-8, n_steps=500):
    """Adam: Adaptive Moment Estimation.
    m = exponentially weighted moving average of gradients (1st moment)
    v = exponentially weighted moving average of squared gradients (2nd moment)
    Bias correction: divides by (1 - beta^t) in early steps when m,v ≈ 0

    Intuition: momentum (m) gives direction, RMSProp (v) gives step size.
    Adam combines both into the best of both worlds.
    """
    w  = w_init.copy()
    m  = np.zeros_like(w)   # 1st moment: mean of gradients
    v  = np.zeros_like(w)   # 2nd moment: mean of squared gradients
    history = [rosenbrock(w)]
    for t in range(1, n_steps + 1):
        g = grad_fn(w)
        m = beta1 * m + (1 - beta1) * g
        v = beta2 * v + (1 - beta2) * g**2

        # Bias correction: early steps have m,v close to 0 (underestimated)
        m_hat = m / (1 - beta1**t)
        v_hat = v / (1 - beta2**t)

        w = w - lr * m_hat / (np.sqrt(v_hat) + eps)
        history.append(rosenbrock(w))
    return w, history

def dropout(x, p_drop=0.5, training=True):
    """
    During training: randomly zero p_drop fraction of neurons.
    During inference: use full network (no dropout).

    Why scale by 1/(1-p_drop)?
    Without scaling: expected activation = (1-p_drop) * x during training.
    At inference (no dropout): activation = x.

    return x * mask / (1.0 - p_drop)    # Scale to maintain expected value

```python
class MLP(nn.Module):
    """Multi-Layer Perceptron with BatchNorm and Dropout."""

    def __init__(self, input_size: int, hidden: list[int], output_size: int,
                 dropout: float = 0.3):
        super().__init__()

        layers: list[nn.Module] = []
        sizes = [input_size] + hidden

        for i in range(len(sizes) - 1):
            layers += [
                nn.Linear(sizes[i], sizes[i+1]),
                nn.BatchNorm1d(sizes[i+1]),   # Normalize layer inputs
                nn.ReLU(),
                nn.Dropout(dropout),          # Random zeroing
            ]

        layers.append(nn.Linear(hidden[-1], output_size))
        self.net = nn.Sequential(*layers)

    def forward(self, x: torch.Tensor) -> torch.Tensor:
        return self.net(x)
```

```python
from torch.utils.data import DataLoader, TensorDataset
import torch.optim as optim

def train(model, train_loader, val_loader, epochs=30):
    optimizer = optim.AdamW(model.parameters(), lr=1e-3, weight_decay=1e-4)
    scheduler = optim.lr_scheduler.CosineAnnealingLR(optimizer, T_max=epochs)
    criterion = nn.CrossEntropyLoss()

    best_val_acc = 0.0
    patience_counter = 0

    for epoch in range(epochs):
        # ── Training phase ────────────────────────────────────────────────────
        model.train()  # Enables Dropout and BatchNorm training mode
        train_loss = 0.0

        for X_batch, y_batch in train_loader:
            X_batch, y_batch = X_batch.to(device), y_batch.to(device)

            optimizer.zero_grad()              # Step 1: clear old gradients
            logits = model(X_batch)            # Step 2: forward pass
            loss = criterion(logits, y_batch)  # Step 3: compute loss
            loss.backward()                    # Step 4: backprop
            nn.utils.clip_grad_norm_(model.parameters(), max_norm=1.0)  # Clip
            optimizer.step()                   # Step 5: update weights

            train_loss += loss.item() * len(X_batch)

        # ── Validation phase ──────────────────────────────────────────────────
        model.eval()  # Disables Dropout; BatchNorm uses running stats
        correct = total = 0

        with torch.no_grad():
            for X_batch, y_batch in val_loader:
                X_batch, y_batch = X_batch.to(device), y_batch.to(device)
                logits = model(X_batch)
                preds  = logits.argmax(dim=1)
                correct += (preds == y_batch).sum().item()
                total   += len(y_batch)

        val_acc = correct / total
        scheduler.step()   # Adjust learning rate

        # Early stopping
        if val_acc > best_val_acc:
            best_val_acc = val_acc
            torch.save(model.state_dict(), "best_model.pt")
            patience_counter = 0
        else:
            patience_counter += 1
            if patience_counter >= 7:
                print(f"Early stopping at epoch {epoch}")
                break

        print(f"Epoch {epoch+1:2d}: loss={train_loss/len(train_loader.dataset):.4f}"
              f"  val_acc={val_acc:.4f}  lr={scheduler.get_last_lr()[0]:.6f}")

    # Load best weights
    model.load_state_dict(torch.load("best_model.pt"))
    return model
```

def conv2d(image, kernel, padding=0, stride=1):
    """
    image:  (H, W) — grayscale image
    kernel: (kH, kW) — convolution filter
    Returns: feature map after convolution

    Each output pixel = sum of element-wise multiply of kernel with image patch.
    The kernel slides across the image, detecting the pattern it encodes.
    """
    H, W = image.shape
    kH, kW = kernel.shape

    # Pad image with zeros
    if padding > 0:
        image = np.pad(image, padding, mode="constant")
        H, W = image.shape

    # Output dimensions
    out_H = (H - kH) // stride + 1
    out_W = (W - kW) // stride + 1
    output = np.zeros((out_H, out_W))

    for i in range(0, out_H):
        for j in range(0, out_W):
            patch = image[i*stride:i*stride+kH, j*stride:j*stride+kW]
            output[i, j] = (patch * kernel).sum()

    return output

class Model:
    def __init__(self, sizes, lr=0.01, l2=0.001):
        rng = np.random.default_rng(42)
        self.lr = lr
        self.l2 = l2    # Weight decay coefficient
        self.layers = []
        for i in range(len(sizes)-1):
            scale = np.sqrt(2.0 / sizes[i])
            self.layers.append({
                "W": rng.normal(0, scale, (sizes[i], sizes[i+1])),
                "b": np.zeros(sizes[i+1]),
                "is_output": i == len(sizes)-2
            })

    def sigmoid(self, x):
        return 1 / (1 + np.exp(-np.clip(x, -500, 500)))

    def forward(self, X):
        self._cache = {"X": X, "zs": [], "as": [X]}
        A = X
        for layer in self.layers:
            Z = A @ layer["W"] + layer["b"]
            self._cache["zs"].append(Z)
            if layer["is_output"]:
                A = self.sigmoid(Z)
            else:
                A = np.maximum(0, Z)  # ReLU
            self._cache["as"].append(A)
        return A

    def loss(self, y_pred, y_true):
        y_pred = np.clip(y_pred[:, 0], 1e-8, 1-1e-8)
        bce = -np.mean(y_true * np.log(y_pred) + (1-y_true) * np.log(1-y_pred))
        # L2 regularization: add sum of squared weights (not bias)
        l2_penalty = sum(np.sum(l["W"]**2) for l in self.layers)
        return bce + self.l2 * l2_penalty / 2

    def backward(self, y_true):
        m = len(y_true)
        dA = self._cache["as"][-1] - y_true.reshape(-1, 1)

        for i in reversed(range(len(self.layers))):

            A_prev = self._cache["as"][i]
            layer = self.layers[i]

            else:
                dZ = dA * (Z > 0).astype(float)

            dW = A_prev.T @ dZ / m + self.l2 * layer["W"]  # L2 gradient
            db = dZ.mean(0)
            dA = dZ @ layer["W"].T

            layer["W"] -= self.lr * dW
            layer["b"] -= self.lr * db

```python
import torch
import torch.nn as nn
import torchvision

class ConvBlock(nn.Module):
    """Conv → BatchNorm → ReLU → optional MaxPool"""
    def __init__(self, in_channels, out_channels, pool=False):
        super().__init__()
        layers = [
            nn.Conv2d(in_channels, out_channels, kernel_size=3, padding=1),
            nn.BatchNorm2d(out_channels),
            nn.ReLU(inplace=True),
        ]
        if pool:
            layers.append(nn.MaxPool2d(2))
        self.block = nn.Sequential(*layers)

    def forward(self, x):
        return self.block(x)

class CNN(nn.Module):
    """CNN for CIFAR-10 (32x32 RGB images, 10 classes)."""

    def __init__(self, n_classes=10):
        super().__init__()
        self.features = nn.Sequential(
            ConvBlock(3,  64,  pool=False),  # (3,32,32) → (64,32,32)
            ConvBlock(64, 64,  pool=True),   # → (64,16,16)
            ConvBlock(64, 128, pool=False),  # → (128,16,16)
            ConvBlock(128,128, pool=True),   # → (128,8,8)
            ConvBlock(128,256, pool=False),  # → (256,8,8)
            ConvBlock(256,256, pool=True),   # → (256,4,4)
        )
        self.classifier = nn.Sequential(
            nn.AdaptiveAvgPool2d(1),         # → (256,1,1) — handles any input size
            nn.Flatten(),                    # → (256,)
            nn.Linear(256, 128),
            nn.ReLU(),
            nn.Dropout(0.5),
            nn.Linear(128, n_classes),
        )

    def forward(self, x):
        return self.classifier(self.features(x))

# Transfer learning: fine-tune ResNet-18
def make_finetune_model(n_classes=10, freeze_backbone=True):
    model = torchvision.models.resnet18(weights="DEFAULT")

    if freeze_backbone:
        for param in model.parameters():
            param.requires_grad = False

    # Replace the final fully connected layer
    in_features = model.fc.in_features    # 512 for ResNet-18
    model.fc = nn.Sequential(
        nn.Linear(in_features, 256),
        nn.ReLU(),
        nn.Dropout(0.4),
        nn.Linear(256, n_classes),
    )
    return model
```
