Data analysis is the backbone of every data science project. Before you can build models or generate dashboards, you need to understand your data — where it comes from, what shape it takes, where it breaks, and what stories it tells. This course teaches you to do exactly that using Python's industry-standard data stack.

You will move from environment setup and Python fundamentals all the way to conducting a full exploratory data analysis on a real dataset. Each lesson builds on the last: NumPy gives you the numerical foundation, Pandas gives you the table-oriented interface, and Matplotlib and Seaborn turn numbers into insight.

No prior data science experience is required, but you should be comfortable writing basic Python (loops, functions, conditionals). By the end, you will be able to take a raw CSV file and produce a coherent, well-visualised analysis report entirely in Python.
