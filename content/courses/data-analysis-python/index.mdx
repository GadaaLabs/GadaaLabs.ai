This is not an introduction to pandas. It is not a tutorial on how to load a CSV file or draw a bar chart. This course is the definitive practitioner track for analysts and data professionals who want to work the way the best in the field actually work — rigorously, systematically, and with a clear chain of logic from a stakeholder question all the way to a delivered recommendation. Every lesson reflects real patterns from production analytical work, not toy examples dressed up to look educational.

The course follows the complete analytical project lifecycle. You begin before any code is written: learning how to interrogate a vague business problem, frame a precise analytical question, and set up the constraints and hypotheses that will govern every decision downstream. You then move through industrial-strength data ingestion and profiling, systematic quality auditing, advanced cleaning strategies, and the complex transformations — reshaping, windowing, time-series joins — that separate professional work from beginner notebooks. Exploratory data analysis is treated not as casual browsing but as a structured scientific process with a protocol, a hypothesis log, and documented findings at every stage.

The second half of the course shifts into the higher-order skills that distinguish a data analyst from a data professional. You will engineer features with intent, applying domain knowledge to create signals that carry meaning. You will apply the right statistical test to the right problem — not just t-tests by default, but the full toolkit of parametric and non-parametric methods, A/B testing frameworks, multiple-comparison corrections, and effect size reasoning. Visualisation is covered as a communication discipline, not a styling exercise: you will learn chart selection theory, annotation strategy, and how to build reusable plotting modules that enforce consistency across a team. The final lessons treat insight generation and professional reporting as first-class analytical skills, covering the Pyramid Principle, the SCR narrative framework, executive summary writing, and notebook-to-deliverable workflows.

Throughout every lesson, the fictional GadaaLabs e-commerce dataset — orders, customers, products, events — provides a consistent real-world context. Code is production-quality: type-hinted helper functions, reusable pipelines, dataclasses for structured outputs, and patterns you can drop directly into your own work. By the end of this track, you will have a complete analytical toolkit and, more importantly, the professional judgment to know when and how to use every part of it.
